# Scripts Index

This directory is the canonical home for executable project scripts.

## Mandatory structure
- Each script must live in its own subdirectory.
- Each script subdirectory must include a `README.md` with usage, examples, and outputs.

## Available scripts
- `generate_clip_chain/` - Runs the three pipeline primitives (hunt, crop, glitch) and the dev composition mode.
- `tests/` - Test suite for pipeline scripts.
